## Instrucciones de Ejecución
1. Asegúrate de tener Node.js instalado (versión 16 o superior).
2. Clona el repositorio: `git clone https://github.com/Prezs2/Backend-Supermercado.git`
3. Instala dependencias: `npm install`
4. Configura la base de datos en `config/database.js`.
5. Ejecuta el servidor: `npm start`
6. El servidor estará disponible en `http://localhost:3000`.

## Ejemplos de Endpoints
- GET /api/products: Obtiene la lista de productos.
- POST /api/products: Crea un nuevo producto (requiere JSON con nombre, precio, etc.).
- GET /api/products/:id: Obtiene detalles de un producto específico.
- PUT /api/products/:id: Actualiza un producto existente.
- DELETE /api/products/:id: Elimina un producto.
